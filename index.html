Why ?

Lately, AngularJS became a major brick of my projects stack and I love the fact that I can use quick-and-dirty JSON to create my model. However, once the first prototype was done, I had to implement an API that usually was only a CRUD API with some findBy features.

Tired of that situation, I took advantage of a private hackathon organized by some folks at Lateal Throughts to start the project.
